## Run Locally

Clone the project

```bash
  git@github.com:rogigs/microfrontend-cart.git
```

Go to a module the project directory

```bash
  cd microfrontend-cart/[name-module]
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
